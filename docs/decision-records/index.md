## Decision Recordとは

Decision Record（DR）は、プロジェクトにおける重要な決定事項を記録するドキュメントである。
「なぜその選択をしたか」を追跡可能にし、将来の意思決定や新メンバーのオンボーディングに役立てる。

記録すべき判断: 技術選定、アーキテクチャ、運用方針、パターン・規約、インフラ、セキュリティ、外部連携など。

## ステータス凡例

| ステータス | 説明 |
|-----------|------|
| 提案 | 決定案を提案中。レビュー・議論が必要 |
| 承認 | チームで合意し、採用が決定 |
| 廃止 | 以前は有効だったが、現在は適用されない |
| 置換 | 新しいDRによって置き換えられた |
