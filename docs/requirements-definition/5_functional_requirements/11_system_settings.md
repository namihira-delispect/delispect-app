# 11.システム設定

## 機能概要
システム管理者がアプリケーションの動作設定を管理する機能。

## 前提条件
- システム管理者または全権管理者でログインしていること

## 機能要件

### 11.1 バッチインポート設定
日次バッチインポートの動作を設定する。

| 設定項目 | 説明 | デフォルト値 |
|---------|------|------------|
| 実行時刻 | バッチインポートの実行時刻 | 06:00 |
| 対象入院日付範囲 | 実行日から何日前までを対象とするか | 2日前 |

### 11.2 設定変更履歴
- 設定変更時に変更内容・変更者・変更日時を監査ログに記録する

## 制約条件
- 設定変更はシステム管理者または全権管理者に限定される
- 設定変更は即時反映される（次回バッチ実行時から適用）

## エラー処理
- 不正な値が入力された場合、バリデーションエラーを表示する
- 保存に失敗した場合、エラーメッセージを表示する
