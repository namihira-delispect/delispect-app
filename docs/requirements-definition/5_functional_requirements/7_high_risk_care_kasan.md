# 7.せん妄ハイリスクケア加算項目

## 機能概要
せん妄ハイリスクケア加算のアセスメント項目を表示し、各項目について該当するかチェックできる機能。
アセスメント項目に一つでも該当する場合、せん妄ハイリスクケア加算対象となる。

## 前提条件
- 患者の入院情報が登録されていること
- リスク評価が実施されていないこと

## 機能要件

### 7.1 アセスメント項目表示機能
- せん妄ハイリスクケア加算のアセスメント項目を一覧表示
- 各項目の判定基準を表示
- チェックボックスで該当/非該当を選択可能

### 7.2 判定結果表示機能
- チェックした項目に基づく判定結果を表示
  - 一つでも該当項目がある場合、せん妄ハイリスクケア加算対象として表示
  - 該当項目がない場合、非対象として表示
- 該当項目の一覧を表示

### 7.3 加算対象の判定機能
- 電子カルテデータとアセスメント項目による判定
  - 電子カルテから取得した診断名、薬剤情報、年齢等の自動取得
  - 手動入力されたアセスメント項目の統合
  - 判定結果の確認・修正機能
- 判定基準
  - 厚生労働省の基準に基づく既定のロジックを使用
  - 自動取得項目と手動入力項目の組み合わせで判定

## 制約条件
- アセスメント項目は厚生労働省の基準に準拠
- 判定結果は医療記録として保存

## エラー処理
- アセスメント項目の欠損時の処理
  - 必須項目が未入力の場合、エラーメッセージを表示する
- 判定不能時の処理
  - 判定に必要な情報が不足している場合、エラーメッセージを表示する
- 保存エラー時の処理
  - 判定結果の保存に失敗した場合、エラーメッセージを表示する
