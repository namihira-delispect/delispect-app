# 用語

## 医療・臨床用語

| 用語              | 意味                                                              |
|:----------------|:----------------------------------------------------------------|
| せん妄 (Delirium)  | 身体疾患に伴う意識障害の一種。注意力や認知機能が急性に変動する                                 |
| 認知症             | 慢性的な認知機能の低下。せん妄のリスク因子の一つ                                        |
| 脳器質的障害          | 脳の器質的な損傷に起因する障害。せん妄のリスク因子                                       |
| せん妄既往           | 過去にせん妄を発症した履歴。リスク因子として評価に使用                                     |
| 全身麻酔            | 手術時の全身的な麻酔。術後せん妄のリスク因子                                          |
| バイタルサイン         | 体温・脈拍・血圧・SpO2 等の生命徴候                                            |
| SpO2            | 経皮的動脈血酸素飽和度 (Saturation of Percutaneous Oxygen)                 |
| 収縮期血圧 / 拡張期血圧   | 血圧の最高値（心臓収縮時）と最低値（心臓拡張時）                                        |
| 呼吸数             | 1分間あたりの呼吸回数                                                     |
| 血液一般検査（CBC）     | Complete Blood Count。赤血球・白血球・血小板等の血球成分を測定する基本検査。入院前の一般採血検査として実施 |
| RBC             | 赤血球数 (Red Blood Cell)。血液一般検査（CBC）で測定                            |
| WBC             | 白血球数 (White Blood Cell)。血液一般検査（CBC）で測定                          |
| Hb (ヘモグロビン)     | 酸素を運搬する血液中のタンパク質。血液一般検査（CBC）で測定                                 |
| Ht (ヘマトクリット)    | 血液中の赤血球の容積割合。血液一般検査（CBC）で測定                                     |
| Plt (血小板)       | 血小板数 (Platelet)。止血に関与する血球成分。血液一般検査（CBC）で測定                      |
| 生化学検査           | 血液中の化学成分を測定する検査。肝機能・腎機能・電解質・炎症マーカー等を含む。入院前の一般採血検査として実施          |
| AST             | アスパラギン酸アミノトランスフェラーゼ。肝機能の指標。生化学検査で測定                             |
| ALT             | アラニンアミノトランスフェラーゼ。肝機能の指標。生化学検査で測定                                |
| ALP             | アルカリホスファターゼ。肝・胆道系機能の指標。生化学検査で測定                                 |
| γ-GT            | ガンマグルタミルトランスフェラーゼ。肝・胆道系機能の指標。生化学検査で測定                           |
| ChE (コリンエステラーゼ) | 肝機能・栄養状態の指標。生化学検査で測定                                            |
| CRE (クレアチニン)    | 腎機能の指標。生化学検査で測定                                                 |
| UN (尿素窒素)       | 腎機能の指標。BUN (Blood Urea Nitrogen) とも呼ばれる。生化学検査で測定                |
| Na (ナトリウム)      | 電解質。体液バランスの指標。生化学検査で測定                                          |
| K (カリウム)        | 電解質。心機能・筋機能に関与。生化学検査で測定                                         |
| Ca (カルシウム)      | 電解質。骨代謝・神経機能に関与。生化学検査で測定                                        |
| GLU (血糖)        | 血中グルコース濃度。糖代謝の指標。生化学検査で測定                                       |
| CRP             | C反応性タンパク。炎症マーカー。生化学検査で測定                                        |
| リスク薬剤           | せん妄発症リスクを高める薬剤の総称                                               |
| オピオイド           | 麻薬性鎮痛薬。せん妄リスクを高める薬剤の一つ                                          |
| ブリストルスケール       | 便の形状を7段階で分類する国際的な評価尺度                                           |
| 腸蠕動音            | 聴診器で確認する腸の動きの音                                                  |
| 疼痛               | 痛みの医学用語。ケアプランでは部位・程度・生活への影響を評価する                                |
| 離床（早期離床）         | 術後や入院中の患者をベッドから離れて活動させること。せん妄予防に有効                               |
| アルコール多飲          | 習慣的な多量飲酒。せん妄のリスク因子の一つ                                           |
| 向精神薬             | 精神機能に作用する薬剤の総称。せん妄のリスク因子の一つ                                     |
| 看護記録             | 電子カルテ上の看護師による記録。ケアプラン詳細からテキスト転記が可能                              |
| デルマトーム          | 脊髄神経が支配する皮膚領域。疼痛の部位特定に使用                                        |
| DPC             | 診断群分類 (Diagnosis Procedure Combination)。入院医療費の包括評価制度            |

## ドメインモデル用語

| 用語                           | 意味                                                                |
|:-----------------------------|:------------------------------------------------------------------|
| Patient（患者）                  | 電子カルテから取得される患者の基本情報                                               |
| Admission（入院）                | 患者の入院レコード。同一患者が複数回入院する場合がある                                       |
| MedicalHistory（既往歴）          | ユーザーが入力するせん妄リスク因子（認知症・脳器質的障害等）。ML予測と加算判定の両方の入力                    |
| RiskAssessment（リスク評価）    | MLモデルによるせん妄発症リスクの予測結果（高/低/判定不能）。HL7 FHIR の RiskAssessment リソースに対応 |
| CarePlan（ケアプラン）              | せん妄予防のための看護ケア計画                                                   |
| CarePlanItem（ケアプラン項目）        | ケアプランを構成する各項目（薬剤・疼痛・脱水・便秘・炎症・離床・認知症・安全管理・睡眠・情報提供）                   |
| HighRiskCareKasan（ハイリスクケア加算） | 厚労省基準に基づくせん妄ハイリスク患者ケア加算。日本の診療報酬制度に固有の概念のため日本語名を採用                 |
| VitalSign（バイタルサイン）           | バイタル情報のレコード                                                       |
| LabResult（採血結果）              | 採血検査結果のレコード                                                       |
| Prescription（処方）             | 患者への処方記録（行為）。FHIR の MedicationRequest に対応。リスク薬剤との照合に使用            |
| MedicationMaster（薬剤マスタ）      | 薬剤そのものの参照データ（モノ）。FHIR の MedicationKnowledge に対応。リスク薬剤の判定に使用       |
| ReferenceValueMaster（基準値マスタ） | バイタル・採血の正常範囲を管理するマスタデータ                                           |

## システム・技術用語

| 用語          | 意味                                                |
|:------------|:--------------------------------------------------|
| 電子カルテ (EMR) | 病院の既存の電子カルテシステム。データ連携元                            |
| 電子カルテ同期 (EMR Sync) | 病院DBから臨床データ（採血・処方・バイタル等）を取得しシステムに反映する処理。手動実行と日次バッチの2方式がある |
| DWH         | データウェアハウス。病院側のデータ蓄積基盤                             |
| SaMD        | 医療機器プログラム (Software as a Medical Device)          |
| HL7         | 医療情報交換の国際標準規格                                     |
| ML API      | 機械学習によるせん妄発症予測を行うリスク判定サービス                        |
| 特徴量         | 機械学習モデルへの入力パラメータ                                  |
| リスクスコア      | MLモデルが算出するせん妄発症リスクの数値                             |
| 楽観的ロック      | バージョン番号による同時更新の競合検出方式                             |
| 監査ログ        | SaMD要件に基づくシステム操作の記録                               |
| RPO         | 目標復旧地点 (Recovery Point Objective)。許容されるデータ損失の最大時間 |
| RTO         | 目標復旧時間 (Recovery Time Objective)。許容されるダウンタイムの最大時間 |
